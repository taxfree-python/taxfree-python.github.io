自分用にメモ
shape=(x, y, z)のndarrayをy→x→zの順で埋めていくには
```Python
np.sort(np.arange(x*y*z).reshape(z,y,x).T.reshape(x*y, z), axis = 0).reshape(x, y, z)
```

x→y→zの順で埋めていくには
```Python
np.sort(np.arange(x*y*z).reshape(z,y,x).T., axis = 0).reshape(x, y, z)
```
とすればいい．
